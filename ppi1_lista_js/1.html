O que é JavaScript?

O JavaScript é uma linguagem de programação, interpretada para desenvovlvimento web. Ele pode funcionar tanto no navegador, que é o lado do usuário, quanto no servidor, com ferramentas como o Node.js. No navegador, ele roda dentro de um ambiente seguro chamado “sandbox”, que impede o acesso a arquivos pessoais do computador e garante mais segurança. Alguns objetos importantes do JavaScript são: document, que permite alterar o conteúdo da página; console, que mostra mensagens e informações úteis para o desenvolvedor; e alert, que exibe avisos e mensagens na tela.
