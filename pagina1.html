Minha Primeira Página

  • Git init - inicia um novo repositório Git
  • Git add - adiciona arquivos ao staging (preparar para commit)
  • Git status - mostra o status dos arquivos no repositório
  • Git commit - salva as alterações adicionadas (com uma mensagem)
  • Git push - envia os commits para o repositório remoto (ex: GitHub)
  • Git log - mostra o histórico de commits
  • Git branch - lista ou cria branches (ramificações)
  • Git checkout - troca para outro branch ou commit
  • Git checkout -b - cria e troca para um novo branch
