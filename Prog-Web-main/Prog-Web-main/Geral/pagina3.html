O CSS

Data de criação: Proposto em 1994 por Håkon Wium Lie e Bert Bos. A primeira versão oficial, o CSS1, foi lançada em 1996 pelo W3C.

Função: Serve para estilizar páginas web — cor, fonte, layout, animação, tudo que dá vida ao HTML.

Evolução:

  • CSS1 (1996): Estilos básicos como cor, fonte e margens.
  • CSS2 (1998): Posicionamento, modelos de caixa, layouts mais complexos.
  • CSS3 (anos 2000+): Transições, animações, flexbox, grid, seletores avançados.

Pré-processadores: Ferramentas como Sass, Less e Stylus ajudam a organizar e turbinar o CSS.

Importância: Separar o conteúdo (HTML) da aparência (CSS) tornou o desenvolvimento web mais limpo, flexível e profissional.

Uma regra CSS pode ser definida com a seguinte sintaxe: 

        Seletor{
            Propriedade1: valor;
            Propriedade2: valor;
        }

Podemos incluir codigo css de 3 formas:

Css externo
Qando definimos codigo Css por meio de um arquivo externo .css, vinculando ele ao html por meio da tag < link > no head do documento html.
CSS inline
Quando definimos codigo CSS de estilização no proprio elemento HTML como: Texto em cor vermelha
